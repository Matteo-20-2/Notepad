## Notepad

Notepad is a simple and lightweight desktop application written in Java for writing and editing text. It is designed as a basic alternative to the classic Notepad, but with some extra features to enhance the user experience.

## Key Features

* Modern interface powered by FlatLaf
* Open, save, and create new text files
* Real-time character and word counter
* Text zoom using Ctrl + mouse wheel
* Support for `.txt` files
* Compatible with Windows, Linux, and macOS (Java 17+)
