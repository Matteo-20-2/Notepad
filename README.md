## Notepad

Notepad is a simple and lightweight desktop application written in Java for writing and editing text and code. It is designed as a basic alternative to the classic Notepad, but with some extra features to enhance the user experience.

## Key Features

* Modern interface powered by FlatLaf
* Open, save, and create new files
* Syntax Highlighting for code
* Real-time character and word counter
* Text zoom using Ctrl + mouse wheel
* Support for `.txt`, `.java`, `.py`, `.c`, `.cpp`, `.html`, `.css`. `.js`, `.json` 
* Compatible with Windows, Linux, and macOS (Java 17+)
